# Sales Forecast

## Description

Create forecast for sales based on history data

## How To

1. Execute **make build**
2. Execute **make run**
3. Execute **make help** to see all commands.
3. All commands can be found in Makefile. Run it sequentially.
4. If **make migrate-init** error, make sure to delete python file inside db/migrations/versions/. Execute **make migrate-init** after that.
